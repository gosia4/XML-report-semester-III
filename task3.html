Task 3

Four transformation to: svg, txt, xhtml, xml.




Transformations using XSTL

For the XML document from Task 2, create four XSLT transformation templates.

  • The first transformation is to create an auxiliary document in XML format, which will be a report obtained from the source document.
  • The second transformation is to obtain a document for presentation in XHTML from the supporting document, with additional requirements increasing the functionality of the document.
  • The third transformation is to obtain an output document in text format from the supporting document.
  • The fourth transformation is to obtain an SVG document (graph) from the supporting document as another way of presenting the data
