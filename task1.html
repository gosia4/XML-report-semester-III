Task 1

The task include an xml file. It performs four composers, their pieces, which are seperately and books (pieces gathered in groups).

Description

Create an XML document structure (one set of nodes/data) that will be used in the next tasks.
  • document structure must contain
    • elements consisting of only values
    • elements that consist of only subelements
    • elements with only attribute/attributes
    • elements with attribute/attributes and only with simple content (e.g. text, number, date, etc.)
  • the structure of the document must contain two groups of information enabling the reference of one piece of information to another (combining information)
    • example: one set of information contains data about a person, the other one about music genres - in the set of information about a person, we have a connection between a given person and the genre of music that he or she likes (we do not enter the genre directly, we refer to it)
  • the structure of the document must contain various types of data
    • strings with various meanings and lengths (including those for which the test pattern can be specified
    • numbers in various formats
    • date (s), amount of money
    • elements with a specific set of values (enumeration types)
  • The correctness of the created document should be checked in the validator.
